# Practice Python – Step-by-Step Problem Solving
  This repository contains my solutions to Python exercises from **[PracticePython.org](https://www.practicepython.org/)**.
  The problems are solved in increasing order of difficulty and are designed to strengthen Python fundamentals, logical thinking, and problem-solving skills through mini-project–style exercises.
## What This Repository Covers
The solutions in this repository focus on core Python concepts, including:
- Variables and data types
- Conditional statements
- Loops
- Functions
- Lists and dictionaries
- Basic string manipulation
- User input handling
- Simple game and utility programs
## Example Problems Included
Some of the exercises implemented in this repository include:
- Password Generator
- List Overlap
- Rock–Paper–Scissors Game
- Number Guessing Game
- String and List Manipulation Problems
